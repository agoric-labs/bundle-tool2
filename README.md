# Agoric Bundle Explorer

To start in dev mode, follow [vite](https://vitejs.dev/) conventions:

```
git clone https://github.com/agoric-labs/bundle-tool2
cd bundle-tool2
yarn
yarn dev
```

Either

- drop / choose a file
- Paste a devnet tx id, hit explore.
